Start charting in 30 seconds

Add ChartPane to Claude Desktop and restart.

Server URL 
https://mcp.chartpane.com/mcp

Open Claude Desktop → Settings → Connectors → Add custom connector → paste the URL above.

claude_desktop_config.json 
{
  "mcpServers": {
    "chartpane": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://mcp.chartpane.com/mcp"]
    }
  }
}

Add this to your claude_desktop_config.json and restart Claude Desktop. Requires Node.js.
