Start charting in 30 seconds

Works with Claude Desktop, ChatGPT, VS Code, and any MCP-compatible client. Free, no API keys, no credit card.

Server URL 
https://mcp.chartpane.com/mcp

Open Claude Desktop → Settings → Connectors → Add custom connector → paste the URL above.

claude_desktop_config.json 
{
  "mcpServers": {
    "chartpane": {
      "command": "npx",
      "args": ["-y", "mcp-remote", "https://mcp.chartpane.com/mcp"]
    }
  }
}

Add this to your config file and restart your client. Requires Node.js.

Frequently asked questions

Is ChartPane free?

Yes, completely free. No API keys, no credit card, no usage limits.

What data do you collect?

Charts render in your browser. We store chart data so you can access your chart history later. We never store your conversations or personal information.

Which AI clients does it work with?

Claude, Claude Desktop, ChatGPT, VS Code, and other clients that support MCP Apps — including Goose, Postman, and MCPJam.

Can I create dashboards with multiple charts?

Yes — just ask for a dashboard and get multiple charts in one response. You can request any combination of chart types.

Do I need to know how to code?

No. Describe what you want in plain English and ChartPane handles the rest. If you're already asking Claude to analyze data, you already know how to use ChartPane.
